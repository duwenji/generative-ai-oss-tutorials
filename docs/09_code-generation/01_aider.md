# Aider 入門

> 📖 中級（概念・実践） | 前提: Python基礎 / LLMアプリの基本概念

## 概要
Aider は CLI で既存コードを直接編集できるコーディングアシスタントです。小さな修正を高速に繰り返す用途に向きます。

## 詳細
- 複数ファイル編集
- テスト実行と修正ループ

## 最小セットアップ

### インストール

```bash
pip install aider-chat
```

### 実行例

```bash
aider --model gpt-4o-mini src/main.py
```
