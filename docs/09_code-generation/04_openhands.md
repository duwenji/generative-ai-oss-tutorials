> 📖 中級（概念・実践） | 前提: Python基礎 / LLMアプリの基本概念

## 概要
OpenHands は Issue ベースの実装タスクを自律実行する開発エージェントです。要件からコード変更までを試験的に自動化できます。

## 詳細
- タスク解析
- コード編集提案
- 実行ログ付きで検証

## 最小セットアップ

### 前提条件

- Docker
- OpenAI 互換APIキー

### 起動イメージ

```bash
docker run -it --rm \
  -e OPENAI_API_KEY=your-key \
  allhandsai/openhands:latest
```
