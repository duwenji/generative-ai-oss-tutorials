## 最小セットアップ

### 構成例

```yaml
version: "3.8"

services:
	tabby:
		image: tabbyml/tabby:latest
		container_name: tabby
		ports:
			- "8082:8080"
		command: ["serve", "--model", "TabbyML/StarCoder-1B"]
		restart: unless-stopped
```

### 起動

```bash
docker-compose up -d
```

### 接続

- URL: http://localhost:8082
- IDE側で Tabby 拡張を使って接続
