# Chatbot UI シナリオ撮影計画

このファイルは [../05-chatbot-ui.md](../05-chatbot-ui.md) の証跡を取得するための台本です。

## 前提

- ローカルで `npm install` と `npm run dev` が完了している
- `.env.local` に利用 Provider のキーが設定済み

## シナリオ

### 01. 依存解決

- 操作: `npm install` 実行結果を撮影
- 保存: `01-npm-install.png`

### 02. 環境変数設定

- 操作: `.env.local` でキー名が読める位置を撮影（秘密値は隠す）
- 保存: `02-env-local.png`

### 03. 開発サーバ起動

- 操作: `npm run dev` で起動ログを撮影
- 保存: `03-dev-server-started.png`

### 04. ホーム表示

- 操作: ブラウザでトップ画面を表示
- 保存: `04-home.png`

### 05. 送信前入力

- 操作: `こんにちは。3行で自己紹介して。` を入力し送信前撮影
- 保存: `05-chat-input.png`

### 06. 送信後応答

- 操作: 05 から送信して応答表示後に撮影
- 保存: `06-chat-output.png`

## 品質ゲート

1. 05 と 06 は別状態
2. 02 はキー名が読め、値は秘匿
3. 03 は起動完了が分かる
4. run-log の実行結果と矛盾しない
