# ムード適用
カバーは、生成AI の要素がいくつかの学習レイヤーに分かれて理解できる構造として見えるようにします。背景は明るいオフホワイトにし、コーラル・アンバー・バイオレット・ティールの温かいアクセントで、OSS コミュニティの開放感と技術的な深みを両立させます。複数のノードをカードや小さなボックスに分け、接続線で関係性を示します。

# フォント適用
モダンなサンセリフ書体（Segoe UI、Inter系フォント風）を使用します。タイトルはボールドで信頼感を確保し、サブタイトルはレギュラーウェイトで学習パスの複合性（カテゴリ横断）を示します。

# コンポジション
テキストコンポジション:
- タイトルは上部ゾーンに配置し、構造的なコアとして機能する。
- サブタイトルは下に続く形で、学習の広がりを示す。
- テキストは左寄せまたは中央揃えで、技術的な整然さを演出する。

ビジュアルコンポジション:
- メインビジュアルは、エージェント / RAG / 推論 / UI / 評価 を示す 5 つの角丸カード群にする。
- カードは少しずつ揃え、矢印や点線で接続し、要素の関係を見せる。複雑なネットワーク図は避ける。
- 背景は薄いグリッド線と余白を使い、教育書らしい構造性を出す。
- レイアウトは上部 20% をテキスト、下部 80% を構造化された要素群に割り当てる。

レンダリング注記: ノード（丸、六角形、矩形）はクリーンなフラットフィルと適度に厚いストロークアウトラインを使用。接続線（エッジ）は直線または軽い曲線で統一。グラデーションは避け、フラットかつ明確な色面で表現。ノード内の小テキストは可読性を保つ。

タイプ注記: 複数ノードの相互接続を、過度な複雑さを避けたカードレイアウトで表現する。技術的な統合性と学習パスの複合性を両立させる。

パレット注記: warm-creative パレットは生成AI技術の先進性を親しみやすい教育書の雰囲気へと落とし込み、OSSコミュニティの開放感と学習のしやすさを両立させる。
