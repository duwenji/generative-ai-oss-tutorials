## 概要

**InvokeAI** は実務向け機能を備えた画像生成ツールです。モデル管理や履歴管理がしやすく、運用寄りの構成に向きます。

**バージョン**: 最新版 / OSS準拠（2026-05時点）  
**公式ドキュメント**: https://www.invokeai.org/

## 最小セットアップ

### 必須スキル

- Python 基本（3.10以上推奨）
- 仮想環境の操作

### 環境

- Python 3.10+
- pip
- GPU推奨（VRAM 4GB以上）

### インストール

```bash
pip install invokeai
```

### 初期設定

```bash
invokeai-configure
```

### 起動

```bash
invokeai-web
```

ブラウザで http://127.0.0.1:9090 にアクセスします。
